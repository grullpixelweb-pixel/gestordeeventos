# 🏆 Sorteio Pro - Gestor de Sorteios Premium

**Sorteio Pro** é uma aplicação web moderna e elegante projetada para gerenciar sorteios em tempo real com precisão, estilo e segurança administrativa. Ideal para eventos, lives e promoções que buscam uma experiência visual premium.

## 🇧🇷 Descrição do Projeto

Este projeto oferece uma interface intuitiva para configurar e realizar sorteios automáticos. Diferente de sorteios simples, o **Sorteio Pro** integra um temporizador baseado no horário oficial de Brasília e um sistema de bloqueio de segurança para garantir que apenas o administrador tenha controle total sobre o evento.

### 🌟 Principais Recursos

- **Limite de Participantes**: Suporte para até 15 participantes com nomes personalizados.
- **Segurança Administrativa**: Bloqueio de interface via senha para evitar que participantes alterem ou parem o sorteio.
- **Temporizador Inteligente**: Defina o horário de início e término. O sorteio roda automaticamente entre esses horários.
- **Duração Flexível**: Suporta sorteios com duração de até 3 horas.
- **Truque do Troféu**: Sistema de recuperação de emergência (Master Reset) oculto para o administrador.
- **Interface Premium**: Design moderno com suporte para modo escuro, animações suaves e visual "glassmorphism".
- **Multi-idioma**: Suporte completo para Português (PT) e Espanhol (ES).
- **Responsivo**: Totalmente otimizado para celulares e computadores.

## 🛠️ Tecnologias Utilizadas

- **React.js**: Biblioteca principal para a interface.
- **Vite**: Ferramenta de build ultra-rápida.
- **Lucide React**: Ícones modernos e consistentes.
- **Framer Motion**: Animações fluidas e interações vivas.
- **Vanilla CSS**: Estilização personalizada e alto desempenho.

## 🚀 Como Executar o Projeto

1. Clone o repositório:
   ```bash
   git clone https://github.com/grullpixelweb-pixel/gestordeeventos.git
   ```

2. Instale as dependências:
   ```bash
   npm install
   ```

3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

4. Acesse no seu navegador: `http://localhost:5173`
